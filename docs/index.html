http233-router-go

High-performance HTTP router for Go with a Gin-like API. Radix-tree matching, middleware chains, route groups, and full net/http compatibility.

Install

go get github.com/neko233-com/http233-router-go@v0.1.0

Quick Example

package main

import (
    "net/http"
    http233 "github.com/neko233-com/http233-router-go"
)

func main() {
    r := http233.New()
    r.Use(http233.Recovery())

    r.GET("/", func(c *http233.Context) {
        c.String(http.StatusOK, "Hello, World!")
    })

    r.GET("/users/:id", func(c *http233.Context) {
        c.JSON(http.StatusOK, map[string]string{"id": c.Param("id")})
    })

    http.ListenAndServe(":8080", r.Handler())
}

Features

Fast Routing

Optimized radix tree with static, :param, and *wildcard segments. Context pooling on the hot path.

Gin-like API

Middleware chain, route groups, Context helpers — familiar patterns without pulling in Gin.

Production Ready

405/404 handlers, trailing slash redirect, static files, optional hot reload via fsnotify.

Tested

100+ integration routes, concurrent access tests, automated httprouter benchmark validation.

Why http233?

Most Go projects need more than bare routing: middleware, groups, JSON helpers, and error recovery. http233-router-go delivers Gin-style ergonomics while keeping routing performance competitive with dedicated routers like httprouter.
