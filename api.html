Core types and methods. Full godoc: pkg.go.dev

Router

MethodDescription
New()Create a new router instance.
Use(mws ...HandlerFunc)Register global middleware.
Group(prefix, mws...)Create a route group.
GET/POST/PUT/DELETE/HEAD/OPTIONS/PATCHRegister method handlers.
ANY(path, handler)Register all standard HTTP methods.
Handler()Return http.Handler for ListenAndServe.
HandleMethodNotAllowed()Return 405 when path exists but method does not.
RedirectTrailingSlash(bool)Enable trailing slash redirects.
SetNotFoundHandler / SetNoMethodHandlerCustom 404 / 405 handlers.
SetOnError(fn)Post-chain error callback.
Static / StaticFS / StaticFileStatic file serving.
EnableHotReload / DisableHotReloadFile watcher for dev assets.

RouterGroup

Same HTTP method registrars as Router, plus Use(), nested Group(), and group-scoped Static / StaticFS.
